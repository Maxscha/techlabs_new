## 🚀 Quick start

We use Gatsby.js to build our Site

1. Clone the Repo
2. Install the Gatsby CLI `npm install -g gatsby-cli`
3. Run `npm install`
4. Run `gatsby develop` to start the project locally

## Publish Changes 

1. Run `npm deploy`
2. Wait for the changes appearing.;-)
